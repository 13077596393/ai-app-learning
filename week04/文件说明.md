main.py

负责 API 接口层

services/
负责业务逻辑层

    document_parser.py
    负责文档解析

    text_splitter.py
    负责文本切分

    embedding_service.py
    负责调用 embedding API

    llm_service.py
    负责调用大模型 API

    retrieval_service.py
    负责 pgvector 检索和 top chunks 排序

    rag_service.py
    负责 RAG prompt、context、引用校验、历史上下文、Prompt Injection 检测、上下文压缩

    eval_service.py
    负责 RAG 评测、命中判断、bad case notes、评测结果保存

models.py
负责数据库表结构

schemas.py
负责请求体和响应体
